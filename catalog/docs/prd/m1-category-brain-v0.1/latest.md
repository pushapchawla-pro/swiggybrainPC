### Signal Fusion Formula (v0.5)
```
emerging_score = (
    0.25 x mom_search_growth_norm +
    0.20 x mom_sales_growth_norm +
    0.15 x trend_stability_score +
    0.20 x demand_gap_norm +
    0.10 x s2c_events_norm +
    0.10 x (100 - zero_result_rate)
)
```

### Selection Criteria
- Volume gate: >=3,000 weekly searches
- Growth gate: >=20% MoM search growth OR >=15% MoM sales growth
- Trend gate: Growth in 2+ of 3 months
- Terms ranked by emerging_score, top 20 selected
